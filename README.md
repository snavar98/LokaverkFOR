# LokaverkFOR
Lokaverkfnið mitt mun snúast um söluskála. Manneskjan sem opnar forritið mun geta pantað mat t.d. hamborgara. Svo fær sá aðili upp ákveðið verð og eftir að það er búið að
borga þá fær manneskjan upp kvittun og miða með númeri. Er búinn að gera útlit fyrir forrit. er að reyna að fá borgun til að virka. búinn með gagnagrunn.
